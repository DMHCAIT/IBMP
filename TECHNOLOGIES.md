# Technologies Used

This project uses the following technologies and libraries:

- Next.js 14 (framework for React; server-side rendering, routing, and build)
- React 18
- TypeScript
- Tailwind CSS (utility-first styling)
- PostCSS
- Node.js & npm (runtime and package manager)
- Supabase (`@supabase/supabase-js`) for backend/auth/data
- Framer Motion (animations)
- Lottie React (animations)
- Lucide React (icons)
- ESLint (linting)

Dev tooling and config:

- `next` scripts: `dev`, `build`, `start`
- `typescript`, `@types/*` for typing
- `eslint-config-next`

Notes:

- See [package.json](package.json) for exact versions and scripts.
